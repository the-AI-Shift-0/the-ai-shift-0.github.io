Filming Guide - Sai Only

Prompt Builder Demo

This demo requires a local server. Follow the setup steps below before filming.

1

One-Time Setup - Clone the repo

Open your terminal and run:

gh repo clone the-AI-Shift-0/the-ai-shift-0.github.io
Only do this once. After cloning, just open the folder each time you film.
2

One-Time Setup - Start the server via Claude Code

Open the cloned folder in Claude Code. Apurv will send you a message to paste into Claude Code - it contains the setup instructions and credentials. Paste it and hit Enter.

Claude Code will install dependencies, create the credentials file, and start the server automatically. You only need to do this once - after that, just run step 3 below each time.
3

Every filming session - Start the server

After the first setup, just run this in your terminal each time you film:

python3 server/prompt-builder-server.py

Then open the demo in your browser:

http://localhost:5000
4

Running the demo - Step by step

  • Screen 1 (Main page): Show the input area. Click the microphone button and speak your task. Example: "Write a follow-up email to a client who hasn't responded in two weeks."
  • Let it transcribe: After speaking, the voice gets converted to text automatically. You'll see it appear in the text box.
  • Click "Build my prompt": The AI analyzes your task and builds an optimised prompt using prompting best practices.
  • Screen 3 (Results): Show the final structured prompt. Point out how it's broken into clear sections - role, context, task, format.
  • Copy and use it: Paste the generated prompt into Claude, ChatGPT, or any AI tool. Show the output you get vs a basic prompt.
Suggested opening line
"Most people just type a one-liner into ChatGPT and wonder why the output is average. This tool takes what you actually want to do - in plain English - and turns it into a proper prompt that gets you 10x better results."
5

Filming tips

Use a real example

Pick a business task that sounds like something an SMB owner actually does - sales follow-up, a refund policy email, a job post.

Show the before/after

Run the same task as a basic one-liner in ChatGPT, then show the output with the generated prompt. The contrast sells it.

Keep voice recording short

10-15 seconds is enough. You don't need a perfect sentence - just a clear task description.

Server must be running

GitHub Pages fallback uses static data. For the voice button and live AI to work, the local server must be running at localhost:5000.
