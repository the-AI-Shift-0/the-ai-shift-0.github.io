Filming Notes & Troubleshooting

Overall Timing

  • Master Trove intro & walkthrough: 45-55 seconds
  • Live demo (index → l2 → l3): 15-20 seconds
  • Closing & CTA: 12-15 seconds
  • Total Demo 2 segment: 75-90 seconds

Key Filming Tips

  • Master Trove must load fast: If the page takes >2 seconds to load, script around it or pre-load it before recording.
  • Show both tabs: Viewers need to see the form (briefly) and the output. Switching tabs makes this seamless.
  • Narrate the output sections: Don't just scroll silently. Explain what each block does and why it matters.
  • Live demo can be fast: The index → l2 → l3 flow doesn't need to be slow. 4-5 seconds per page is fine. Viewers get the idea.
  • Pause on success state: The l3.html page is your payoff moment. Let it breathe for 3-4 seconds so viewers absorb the impact.

If Master Trove Form Fails

If clicking "Next" or "Generate" doesn't work during recording, just skip to the "See Example Output" tab directly. This is the safer route anyway and aligns with Demo 2 best practices.
